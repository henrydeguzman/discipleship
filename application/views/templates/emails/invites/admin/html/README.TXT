1. [ new-email-user.html ]
     - Use this template to send email to recepient assigning him/her as administrator of said center. 
     - system will create an account after he/she accept your invitation.
     - After the creation of account, system will auto email the recepient to send his/her credentials
     using the [ new-account.html ] template
2. [ existing-user.html ]
     - Use this template to send email to recepient that have already an account.
     - She/he needs to accept the invitation before the system will update his/her account to make as an administrator.
3. [ email-accepted.html ]
     - after #1 or #2, use this email to notify the sender about his/her requested invitation from recepient.
